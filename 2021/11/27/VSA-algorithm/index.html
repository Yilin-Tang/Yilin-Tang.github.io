VSA algorithm

Posted on 2021-11-27 Edited on 2021-11-28 In Graphics

Variational Shape Approximation(VSA) 是一种曲面简化算法。它在给定简化程度（使用多少个多边形面）的条件下，最小化与原曲面误差的。该算法分为计算近似多边形面（Proxy）和重建曲面两个部分。

算法背景：k-means算法

k-means算法步骤：

  1. 初始化：随机找k个中心
  2. Partitioning：将N个点分配到它距离最近的中心
  3. Fitting：根据分配结果，计算每一类的平均值作为新的中心
  4. Repeat：从2开始重复，迭代一定步数后停止

k-means算法实际上优化的损失函数是

∑i=1k∑Xj∈Ri∥Xj−ci∥2\sum_{i=1}^{k}\sum_{X_j\in \mathcal{R}_i} \|X_j-c_i\|^2 i=1∑k​Xj​∈Ri​∑​∥Xj​−ci​∥2

其中kkk是分类数，Ri\mathcal{R}_iRi​是划分的区域，cic_ici​是Ri\mathcal{R}_iRi​的中心，XjX_jXj​是点。

VSA受到k-means算法的启发，将曲面简化过程分为Geometry Partitioning和Proxy Fitting两部分，对应于k-means算法中的Partitioning和Fitting。但曲面简化显然不是对三角形做简单聚类，三角形地分类需要保留几何信息，所以需要用到不同的损失函数，也就是L2\mathcal{L}^2L2或L2,1\mathcal{L}^{2,1}L2,1指标。

VSA算法的核心概念

Proxy：向量对P=(X,N)P=(X,\mathbf{N})P=(X,N)，其中XXX是点的坐标，N\mathbf{N}N是法向量。可以将PPP想象为过点XXX、法向量为N\mathbf{N}N的平面

L2\mathcal{L}^2L2：L2(Ri,Pi)=∬x∈Ri∥x−Πi(x)∥2dx\mathcal{L}^{2}(\mathcal{R}_i, P_i)=\iint_{x \in \mathcal{R}_i}\|x-\Pi_i(x)\|^{2} dxL2(Ri​,Pi​)=∬x∈Ri​​∥x−Πi​(x)∥2dx

  1. 初始化（Initial Seeding）：

    随机确定S\mathcal{S}S上的kkk个种子三角形STi(i=1,…,k)ST_i(i=1,\dots,k)STi​(i=1,…,k)，对STiST_iSTi​分配Pi=(Xi,Ni)P_i=(X_i,\mathbf{N}_i)Pi​=(Xi​,Ni​)，其中XiX_iXi​是TiT_iTi​的重心坐标，Ni\mathbf{N}_iNi​是TiT_iTi​的法向量。

  2. Distortion-minimizing Flooding：从种子三角形”生长“出分割区域，即从STiST_iSTi​出发，将S\mathcal{S}S的所有三角形划分到STiST_iSTi​对应的Proxy

    1. 设置一个全局最小优先队列，队列元素为(Tj,Pi)(T_j,P_i)(Tj​,Pi​)，优先度为E(Tj,Pi)E(T_j,P_i)E(Tj​,Pi​)。同时给每个三角形TjT_jTj​设置一个**被分配到哪个PiP_iPi​**的标识。STiST_iSTi​分配给PiP_iPi​。
    2. 初始化队列：STiST_iSTi​相邻的三个三角形(Tij,Pi)(j=0,1,2)(i=1,…,k)(T_{ij},P_i)(j=0,1,2)(i=1,\dots,k)(Tij​,Pi​)(j=0,1,2)(i=1,…,k)入队。

    1. 若E=L2E=\mathcal{L}^2E=L2，则XiX_iXi​是Ri\mathcal{R}_iRi​中所有三角形的重心坐标的加权平均，权重是三角形的面积；Ni\mathbf{N}_iNi​是Ri\mathcal{R}_iRi​的协方差矩阵的最小特征值的特征向量。
    2. 若E=L2,1E=\mathcal{L}^{2,1}E=L2,1，则Ni\mathbf{N}_iNi​是Ri\mathcal{R}_iRi​XiX_iXi​是Ri\mathcal{R}_iRi​中所有三角形的重心坐标的加权平均，权重是三角形的面积；XiX_iXi​与L2,1L^{2,1}L2,1无关，这里取为L2\mathcal{L}^2L2的XiX_iXi​。

  4. 确定新的中心：根据更新后的PiP_iPi​确定新的种子三角形STi(i=1,…,k)ST_i(i=1,\dots,k)STi​(i=1,…,k)

    2. 将STiST_iSTi​分配给PiP_iPi​，同时抹除其余三角形与Proxy的分配关系。
    3. 返回步骤2，直到迭代了给定的次数。

提取网格

VSA算法到此为止得到了分割R\mathcal{R}R和kkk个拟合好的Pi=(Xi,Ni)P_i=(X_i,\mathbf{N}_i)Pi​=(Xi​,Ni​)，但还没有得到一个有效的网格表示。下面介绍如何提取网格。

Anchor Vertices

现在已经找到了近似得最好的Proxies，而Proxies可以看作近似平面，接下来就要在Proxies的平面相交处添加顶点。

  1. 在S\mathcal{S}S的顶点中寻找连接3个或以上区域的顶点，这些顶点会在新网格中生成称为anchor vertex的顶点
  2. 计算这些顶点在它连接的所有PjP_jPj​上的投影，并取这些投影的平均值，这就是该顶点对应的anchor vertex（可理解为原顶点在新网格中的坐标）
  3. 为了保证每个区域至少有一个多边形，需要检查区域Ri\mathcal{R}_iRi​的边界是否有至少三个anchor vertex，不够的话则需要添加。

Edge Extraction

每个anchor vertex与原顶点是对应的，可以通过访问区域的边界将anchor vertices连接起来。但这种方法过于粗糙，会造成几何结构的不精确，所以提出了弧长细分递归算法。

设a\mathbf{a}a和b\mathbf{b}b是两个anchor vertex，连接它们的边分开了PiP_iPi​和PjP_jPj​

  1. 访问a\mathbf{a}a和b\mathbf{b}b之间的区域边界上的原顶点，找到使得点到边(a,b)(\mathbf{a},\mathbf{b})(a,b)的距离ddd最大的点c\mathbf{c}c。

  2. 计算条件

    cond=d⋅sin⁡(Ni,Nj)∥(a,b)∥\text{cond}=d\cdot\frac{\sin(\mathbf{N}_i,\mathbf{N}_j)}{\|(\mathbf{a},\mathbf{b})\|} cond=d⋅∥(a,b)∥sin(Ni​,Nj​)​

    若cond\text{cond}cond小于某个阈值，则不进行任何操作

    若cond\text{cond}cond大于某个阈值，则c\mathbf{c}c处添加一个anchor vertex，然后在边(a,c)(\mathbf{a},\mathbf{c})(a,c)和边(c,b)(\mathbf{c},\mathbf{b})(c,b)重复执行以上操作，直到cond\text{cond}cond小于阈值。

Triangulation

经过上面的步骤后，已经得到了一个多边形网格，但是还无法保证这些多边形是平的或者凸的，因此还需要做三角划分（OpenGL只能渲染三角形网格）。这一过程对每个区域用一个伪CDT（Constrained Delaunay Triangulation）实现，Constrained的意思是保证连接anchor vertices的边在最终生成的三角形网格中。

  1. 给一个区域的多边形顶点（也就是anchor vertices）不同的颜色
  2. 遍历该区域边界的所有原顶点，顶点着上与之最近的anchor vertex的颜色。
  3. 遍历该区域内的所有原顶点，同样地，顶点着上与之最近的anchor vertex的颜色。
  4. 若输入曲面S\mathcal{S}S的三角形的三个顶点有三种不同颜色，则将这三种颜色对应的anchor vertices连接起来生成一个三角形，这些生成的三角形组成最终的三角形网格。

若只是希望得到三角形网格，则VSA算法到这里就完成了。接下来还可以去掉三角形网格中一些冗余的边，得到更为简洁的多边形网格。
